== Description ==
WP Instagram is a WordPress Instagram Feed / Widget Plugin. Create Instagram Feed in a preferred style and place it to your website in any section easily.

This plugin includes Elementor Addon, WP Bakery Page Builder addon, WordPress Widget, 3 Custom styles with different customization option, Slider enable/disable option and column control option.

== Installation ==
This section describes how to install the WP Instagram Plugins for WordPress get it working.

= 1) Install =

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WP Instagram".
3. Install, then Activate it.

= Old Way: =
1. Unzip (if it is zipped) and Upload `ht-instagram` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. Go to the WP Instagram Settings Options.
3. Give your Instagram Access token in instagram access token option.
4. Place your Instagram Feed any position on your site, relax!
